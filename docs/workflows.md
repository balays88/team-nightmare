# Workflows Reference

The Security Suite implements the following core workflows:

---

## Master Security Workflow

**Primary Agent:** Alastor Moody

**Purpose:**
Coordinates all agents in a sequence. Morticia finds the holes, Moody builds the walls, Barnaby checks the locks, and Sherlock tries to pick them.

---

## Pre-Mortem Analysis

**Primary Agent:** Morticia Addams

**Purpose:**
To find the "fatal flaw" in a design before it's too late.

**Key Steps:**

1. The Wake (Imagine failure)
2. The Ghosts (Analyze causes)
3. The Eulogy (Report)

---

## Compliance Audit

**Primary Agent:** Barnaby

**Purpose:**
Static analysis of code and configuration against OWASP Top 10 and GDPR.

---

## Penetration Test

**Primary Agent:** Sherlock Holmes

**Purpose:**
Active simulation of known exploit methods to find bypasses in the architecture.
