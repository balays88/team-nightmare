# Contributing to Security Suite (Nightmare Team)

Thank you for your interest in hardening this security module!

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Ensure all agent personas and workflows maintain the established character styles (Moody, Morticia, Barnaby, Sherlock).
4. Submit a Pull Request with a clear description of your changes.

## Security Note
If you find a vulnerability in this security module itself, please report it via the Issues section with the `security` tag.

Stay Paranoid. Constant Vigilance!
